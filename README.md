
# TicTacToe (OpenGL C++)

A simple Tic-Tac-Toe game built using:

- C++
- OpenGL (GLFW + GLAD)
- Custom shader rendering

## Features
- Mouse click input
- Turn-based gameplay (Blue vs Red)
- Winner detection
- Board reset with **R key**

## Controls
- Left click → place mark
- R → reset game
- Q → quit
